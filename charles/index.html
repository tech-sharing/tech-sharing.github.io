Charles monitorn app

2015-06-17

Charles can monitor the request,response and other more information.

1.Install Charles on osx and get the mac ip address

192.168.1.xxx

2.setting the network on mobile

2.1 iOS

under WiFi setting
HTTP PROXY
Manual
Server: 192.168.1.xxx
Port: 8888(default,you will set other if you change)

2.2 Android
long press the WiFi then selete the modify network
selete Advanced options
Manual
proxy hostname: 192.168.1.xxx
proxy port: 8888(default,you will set other if you change)

3.start recording and run app

you will get the request,response and other more information.

4.Advance for SSL datas (HTTPS)

4.1 download ssl certificate

http://charlesproxy.com/charles.crt

4.2 install the ssl certificate on osx

4.3 install the ssl certificate on mobile(ios/android)

4.4 in Charles Proxy-Setting

Enable SSLProxying
add *.xxx (xxx:port)
